## 核心变更

### Go 后端 (6 机制)
- **流中断恢复**: stream() 返回 interrupted 标志，保存部分输出 + 恢复提示
- **空回答检测**: 模型返回无文本时重试 3 次
- **语言偏好分离**: 推理/回答语言通过瞬态注入，不破坏缓存前缀
- **用户中途转向**: Steer Queue 运行时队列注入引导消息
- **最终就绪闸门**: complete_step 证据验证阻塞
- **Canonical Todo**: 主机侧 todo 状态机，complete_step 自动推进

### React 前端 (11 组件/增强)
- reasoningDisplay: 流式思维链尾截断
- useEntranceAnimation: GSAP 入场动画 Hook
- messageSelectionCopy: 消息区划选纯文本复制
- ProcessCard: 执行流程 SVG 图标组件
- InlineDiff: 紧凑 unified diff 视图
- ErrorBoundary: React 崩溃防护
- ContextMenu: 右键菜单 (Portal/键盘导航/边缘裁剪)
- CopyButton: Wails runtime 剪贴板 + lazy getText
- Message: 长思考折叠优化 + 流式自动展开
- StatusBar: 双速率 (nowRate+avgRate)
- Transcript: GSAP 平滑滚动 + ResizeObserver

### CSS 设计系统 (4 组 token)
